# Shiki Player

Simple Chrome extension that integrates Kodik video player directly into Shikimori anime pages.

## Features
- Adds "Watch Online" button to anime pages.
- Integrates native Kodik player widget.
- Supports mirrors (shiki.one, shikimori.me).
- Automatic fallback to iframe if widget fails.

## Installation (Developer Mode)
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked**.
5. Select the project folder.
