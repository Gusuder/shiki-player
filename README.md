## üá¨üáß English

**Shiki Player** is a lightweight browser extension that brings the anime watching experience back to Shikimori. It seamlessly integrates the native Kodik video player directly into anime pages on Shikimori and its mirrors.

### ‚ú® Features
*   **Seamless Integration**: Adds a stylish "Watch Online" button directly to the anime description page.
*   **Native Widget**: Uses the official Kodik player widget for the best compatibility (supports seasons, episodes, and translations).
*   **Smart Fallback**: Automatically switches to iframe mode if the JS widget fails to load.
*   **Mirror Support**: Works on `shikimori.one`, `shikimori.me`, and `shiki.one`.
*   **No Bloat**: Pure JavaScript, no external frameworks, extremely lightweight.

### üõ† Troubleshooting
*   **Player not loading?**
    *   Ensure you are not blocking scripts from `kodik.info` / `kodik.biz`.
    *   If you are in a region where Kodik is blocked (e.g., Russia/Ukraine/EU), use a VPN or DPI bypass tools (like GoodbyeDPI) ‚Äî the extension relies on your connection to the video server.
